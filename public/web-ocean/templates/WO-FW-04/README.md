# Personal Trainer Template (WO-FW-04)

## Description
Dynamic trainer template with training packages, success stories, and consultation booking. Ideal for personal trainers and coaches.

## Pages
- index.html (Home)
- programs.html (Programs)
- success-stories.html (Success Stories)
- about.html (About)
- book.html (Book)
- blog.html (Blog)

- Primary: #2563eb (Blue)
- Secondary: #16a34a (Green)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Training Packages
- Client Transformations
- Consultation Booking
- Fitness Tips
- Testimonials
