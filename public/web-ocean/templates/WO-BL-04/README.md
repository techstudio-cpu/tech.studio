# Parenting Blog Template (WO-BL-04)

## Description
Warm parenting blog template with age-based tips, resource library, and community features. Perfect for parenting bloggers.

## Pages
- index.html (Home)
- tips.html (Parenting Tips)
- resources.html (Resources)
- community.html (Community)
- about.html (About)
- contact.html (Contact)

## Color Palette
- Primary: #7c3aed (Violet)
- Secondary: #db2777 (Pink)
- Accent: #f59e0b (Amber)
- Background: #fef3c7 (Warm Light)

## Typography
- Headings: 'Playfair Display', serif
- Body: 'Manrope', sans-serif

## Features
- Age-Based Tips
- Resource Downloads
- Parent Stories
- Community Forum
- Newsletter
