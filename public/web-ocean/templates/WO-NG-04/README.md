# Healthcare NGO Template (WO-NG-04)

## Description
Compassionate healthcare template with program information, patient stories, and support options. Ideal for healthcare non-profits.

## Pages
- index.html (Home)
- programs.html (Programs)
- stories.html (Stories)
- support.html (Support)
- resources.html (Resources)
- contact.html (Contact)

## Color Palette
- Primary: #2563eb (Blue)
- Secondary: #dc2626 (Red)
- Accent: #16a34a (Green)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Healthcare Programs
- Patient Stories
- Support Options
- Resource Library
- Volunteer Opportunities
