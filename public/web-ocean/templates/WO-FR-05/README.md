# Specialty Cuisine Template (WO-FR-05)

## Description
Cultural restaurant template with signature dishes, chef profiles, and cultural recipe blog. Perfect for sushi bars, Italian bistros, etc.

## Pages
- index.html (Home)
- menu.html (Menu)
- chef.html (Chef)
- reservations.html (Reservations)
- gallery.html (Gallery)
- blog.html (Blog)

## Color Palette
- Primary: #9333ea (Purple)
- Secondary: #2563eb (Blue)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Playfair Display', serif
- Body: 'Manrope', sans-serif

## Features
- Signature Dishes
- Chef Bio
- Cultural Recipes
- Photo Gallery
- Online Booking
