# Creative Agency Template (WO-CS-05)

## Description
Bold creative agency template with portfolio showcases, team profiles, and service offerings. Built for creative studios.

## Pages
- index.html (Home)
- work.html (Portfolio)
- services.html (Services)
- team.html (Team)
- blog.html (Blog)
- contact.html (Contact)

## Color Palette
- Primary: #7c3aed (Purple)
- Secondary: #db2777 (Pink)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Portfolio Showcase
- Service Packages
- Team Profiles
- Creative Blog
- Client Testimonials
