# Meal Prep Service Template (WO-FR-04)

## Description
Clean meal prep template with weekly plans, nutritional info, and subscription options. Built for meal prep and health food businesses.

## Pages
- index.html (Home)
- meal-plans.html (Meal Plans)
- nutrition.html (Nutrition)
- subscribe.html (Subscribe)
- success-stories.html (Success Stories)
- faq.html (FAQ)

- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Sora', sans-serif
- Body: 'Inter', sans-serif

## Features
- Weekly Meal Plans
- Nutritional Calculator
- Subscription System
- Health Blog
- Delivery Tracking
