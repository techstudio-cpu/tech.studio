# Community NGO Template (WO-NG-03)

## Description
Warm community template with program showcases, volunteer opportunities, and donation options. Perfect for community organizations.

## Pages
- index.html (Home)
- programs.html (Programs)
- stories.html (Stories)
- volunteer.html (Volunteer)
- donate.html (Donate)
- contact.html (Contact)

## Color Palette
- Primary: #2563eb (Blue)
- Secondary: #16a34a (Green)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Program Showcases
- Community Stories
- Volunteer Signup
- Donation Page
- Resource Guides
