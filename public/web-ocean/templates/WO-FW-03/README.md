# Spa Template (WO-FW-03)

## Description
Tranquil spa template with treatment menus, appointment booking, and wellness philosophy. Perfect for spas and wellness retreats.

## Pages
- index.html (Home)
- treatments.html (Treatments)
- book.html (Book)
- philosophy.html (Philosophy)
- gallery.html (Gallery)
- contact.html (Contact)

## Color Palette
- Primary: #0891b2 (Teal)
- Secondary: #9333ea (Purple)
- Accent: #f59e0b (Amber)
- Background: #f8fafc (Light Gray)

## Typography
- Headings: 'Playfair Display', serif
- Body: 'Manrope', sans-serif

## Features
- Treatment Menu
- Online Booking
- Team Profiles
- Wellness Blog
- Gift Vouchers
