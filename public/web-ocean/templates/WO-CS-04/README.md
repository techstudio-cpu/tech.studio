# Legal Services Template (WO-CS-04)

## Description
Professional legal template with practice areas, attorney profiles, and consultation booking. Perfect for law firms and attorneys.

## Pages
- index.html (Home)
- practice-areas.html (Practice Areas)
- attorneys.html (Attorneys)
- resources.html (Resources)
- consult.html (Consultation)
- contact.html (Contact)

- Accent: #d97706 (Gold)
- Background: #f9fafb (Light Gray)

## Typography
- Headings: 'Playfair Display', serif
- Body: 'Manrope', sans-serif

## Features
- Practice Areas
- Attorney Profiles
- Legal Guides
- FAQ Section
- Consultation Booking
