# Tech Blog Template (WO-BL-03)

## Description
Modern tech blog template with gadget reviews, comparison tools, and tech news sections. Ideal for tech reviewers and enthusiasts.

## Pages
- index.html (Home)
- reviews.html (Gadget Reviews)
- compare.html (Product Comparisons)
- news.html (Tech News)
- tutorials.html (How-To Tutorials)
- about.html (About)

## Color Palette
- Primary: #2563eb (Blue)
- Secondary: #1e293b (Navy)
- Accent: #f59e0b (Amber)
- Background: #f8fafc (Light Gray)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif
- Code: 'Fira Code', monospace

## Features
- Gadget Reviews
- Comparison Tool
- Tech News
- How-To Tutorials
- Newsletter
