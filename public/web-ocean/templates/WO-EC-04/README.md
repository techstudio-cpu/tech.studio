# WO-EC-04: Organic Grocery Store

## Template Name
Green Harvest - Organic Grocery Store

## Category
E-commerce > Organic Grocery

## Pages Included
1. **index.html** - Homepage with hero, product categories, featured products, recipes, sustainability section, newsletter
2. **shop.html** - Product catalog with category filters, product grid, search functionality
3. **recipes.html** - Recipe blog with healthy meal ideas and nutritional info
4. **about.html** - Farm story, organic certification, sustainability commitment
5. **contact.html** - Contact form, store location, delivery zones

## Suggested Use Cases
- Organic grocery stores
- Health food retailers
- Farm-to-table businesses
- Natural food markets
- Sustainable food suppliers

## Color Palette
- Primary: #22c55e (Green)
- Secondary: #16a34a (Dark Green)
- Accent: #f59e0b (Amber)
- Text Dark: #1f2937
- Text Light: #6b7280
- Background: #f8fafc

## Typography
- Headings: Montserrat (sans-serif)
- Body: Inter (sans-serif)

## Features
- Responsive mobile-first design
- Product filtering by category, organic certification, price
- Recipe integration with shopping list generation
- Sustainability badges and certifications
- Subscription box options
- Delivery zone calculator
- Fresh produce calendar

## Customization Notes
- Update brand name in navbar and footer
- Replace product images with actual inventory
- Modify color variables in style.css :root
- Update contact information and delivery zones
- Add actual product data and recipes
- Integrate with e-commerce backend
