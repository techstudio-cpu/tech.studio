# WO-EC-05: Pet Supply Store

## Template Name
Pawfect Supplies - Pet Supply Store

## Category
E-commerce > Pet Supplies

## Pages Included
1. **index.html** - Homepage with hero, pet categories, featured products, care tips, adoption section
2. **shop.html** - Product catalog with pet type filters, product grid, search functionality
3. **care.html** - Pet care blog with tips, guides, and expert advice
4. **adoptions.html** - Local shelter partnerships and adoptable pets
5. **about.html** - Store story, values, team, pet-friendly initiatives
6. **contact.html** - Contact form, store location, customer support

## Suggested Use Cases
- Pet supply stores
- Veterinary clinics
- Animal shelters
- Pet grooming services
- Online pet retailers

## Features
- Responsive mobile-first design
- Pet type filtering (dogs, cats, birds, etc.)
- Care tips blog with expert content
- Adoption partnership section
- Loyalty program integration
- Product reviews and ratings
- Wishlist functionality
- Emergency contact information
