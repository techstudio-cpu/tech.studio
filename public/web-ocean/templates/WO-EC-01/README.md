# WO-EC-01: Fashion Boutique Store

## Template Name
Velvet & Thread - Fashion Boutique

## Category
E-commerce > Fashion Boutique

## Pages Included
1. **index.html** - Homepage with hero, categories, featured products, features, lookbook preview, testimonials, newsletter
2. **shop.html** - Product catalog with sidebar filters, product grid, pagination
3. **about.html** - Brand story, values, team, sustainability commitment
4. **contact.html** - Contact form, location info, FAQ section
5. **lookbook.html** - Style inspiration with curated looks and gallery

## Suggested Use Cases
- Fashion boutiques and clothing stores
- Women's apparel brands
- Accessory shops
- Designer clothing retailers
- Online fashion marketplaces

## Color Palette
- Primary: #c9a87c (Rose Gold)
- Secondary: #f5e6e0 (Blush Pink)
- Accent: #d4a5a5 (Dusty Rose)
- Text Dark: #2d2d2d
- Text Light: #6b6b6b
- Background: #faf8f6

## Typography
- Headings: Playfair Display (serif)
- Body: Inter (sans-serif)

## Features
- Responsive mobile-first design
- Product filtering by category, price, size, color
- Product cards with hover effects
- Lookbook with shoppable looks
- Newsletter subscription
- FAQ accordion
- Animated scroll effects
- Back to top button

## Customization Notes
- Update brand name in navbar and footer
- Replace product images with actual inventory
- Modify color variables in style.css :root
- Update contact information
- Add actual product data and links
- Integrate with e-commerce backend (Shopify, WooCommerce, etc.)
