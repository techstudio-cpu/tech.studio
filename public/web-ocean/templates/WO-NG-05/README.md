# Education NGO Template (WO-NG-05)

## Description
Inspiring education template with scholarship programs, student success stories, and donation options. Built for education-focused non-profits.

## Pages
- index.html (Home)
- programs.html (Programs)
- scholarships.html (Scholarships)
- stories.html (Stories)
- donate.html (Donate)
- contact.html (Contact)

## Color Palette
- Primary: #2563eb (Blue)
- Secondary: #9333ea (Purple)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Education Programs
- Scholarship Info
- Student Stories
- Donation System
- Volunteer Teaching
