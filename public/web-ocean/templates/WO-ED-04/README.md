# Art Workshops Template (WO-ED-04)

## Description
Creative workshop template with class schedules, instructor profiles, and project galleries. Perfect for art schools and creative studios.

## Pages
- index.html (Home)
- workshops.html (Workshops)
- instructors.html (Instructors)
- gallery.html (Gallery)
- register.html (Register)
- blog.html (Blog)

## Color Palette
- Primary: #9333ea (Purple)
- Secondary: #2563eb (Blue)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Playfair Display', serif
- Body: 'Manrope', sans-serif

## Features
- Workshop Schedule
- Instructor Bios
- Student Gallery
- Registration System
- Creative Tips Blog
