# Nutritionist Template (WO-FW-05)

## Description
Health-focused template with nutrition plans, recipe library, and consultation booking. Perfect for nutritionists and health coaches.

## Pages
- index.html (Home)
- plans.html (Plans)
- recipes.html (Recipes)
- resources.html (Resources)
- consult.html (Consult)
- success-stories.html (Success Stories)

- Secondary: #2563eb (Blue)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Sora', sans-serif
- Body: 'Inter', sans-serif

## Features
- Nutrition Plans
- Recipe Library
- Downloadable Guides
- Consultation Booking
- Client Results
