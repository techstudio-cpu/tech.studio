# WO-EC-02: Handmade Craft Store

## Template Name
Artisan Haven - Handmade Craft Store

## Category
E-commerce > Handmade Crafts

## Pages Included
1. **index.html** - Homepage with hero, categories, featured products, process, artisans, testimonials, blog preview
2. **shop.html** - Product catalog with filters
3. **artisans.html** - Artisan profiles and stories
4. **custom.html** - Custom order request form
5. **about.html** - Brand story and values
6. **contact.html** - Contact form and info
