# Personal Blog Template (WO-BL-05)

## Description
Clean personal blog template with featured posts, category archives, and social integration. Perfect for personal bloggers and writers.

## Pages
- index.html (Home)
- blog.html (Blog)
- about.html (About)
- categories.html (Categories)
- contact.html (Contact)

## Color Palette
- Primary: #10b981 (Emerald)
- Secondary: #3b82f6 (Blue)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Sora', sans-serif
- Body: 'Inter', sans-serif

## Features
- Featured Posts
- Category Archives
- Comments Section
- Social Integration
- Newsletter Signup
