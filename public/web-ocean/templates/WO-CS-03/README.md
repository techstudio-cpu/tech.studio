# Marketing Agency Template (WO-CS-03)

## Description
Dynamic agency template with service portfolios, campaign showcases, and pricing packages. Built for marketing agencies.

## Pages
- index.html (Home)
- services.html (Services)
- portfolio.html (Portfolio)
- pricing.html (Pricing)
- blog.html (Blog)
- contact.html (Contact)

## Color Palette
- Primary: #3b82f6 (Blue)
- Secondary: #ec4899 (Pink)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Service Portfolio
- Campaign Showcases
- Pricing Packages
- Marketing Blog
- Client Testimonials
