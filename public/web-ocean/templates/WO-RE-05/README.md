# Property Management Template (WO-RE-05)

## Description
Comprehensive property management template with service offerings, owner resources, and tenant portals. Ideal for property managers.

## Pages
- index.html (Home)
- services.html (Services)
- owners.html (Owners)
- tenants.html (Tenants)
- resources.html (Resources)
- contact.html (Contact)

## Typography

- Body: 'Inter', sans-serif

## Features
- Service Packages
- Owner Portal
- Tenant Resources
- Maintenance Requests
- Property Blog
