# Professional Development Template (WO-ED-03)

## Description
Corporate training template with certification programs, webinar calendars, and enrollment systems. Ideal for professional training providers.

## Pages
- index.html (Home)
- programs.html (Programs)
- certifications.html (Certifications)
- webinars.html (Webinars)
- enroll.html (Enroll)
- resources.html (Resources)

- Secondary: #1e293b (Navy)
- Accent: #f59e0b (Amber)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Training Programs
- Certification Tracks
- Webinar Calendar
- Industry Blog
- Corporate Packages
