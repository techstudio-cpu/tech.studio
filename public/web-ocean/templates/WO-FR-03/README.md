# Food Truck Template (WO-FR-03)

## Description
Vibrant food truck template with location tracking, event calendar, and food gallery. Perfect for mobile food vendors.

## Pages
- index.html (Home)
- menu.html (Menu)
- locations.html (Locations)
- events.html (Events)
- gallery.html (Gallery)
- contact.html (Contact)

## Color Palette
- Primary: #dc2626 (Red)
- Secondary: #f59e0b (Amber)
- Accent: #16a34a (Green)
- Background: #ffffff (White)

## Typography
- Headings: 'Space Grotesk', sans-serif
- Body: 'Inter', sans-serif

## Features
- Location Map
- Event Calendar
- Food Gallery
- Story Page
- Social Feed
